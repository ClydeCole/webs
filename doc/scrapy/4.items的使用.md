# Items 的使用

## 什麼是Items

`items.py` 用於定義爬取數據的結構, 類似於數據模型或實體類

## 定義Item

```python
import scrapy

class 類名(scrapy.Item):
    字段名 = scrapy.Field()
```

- 繼承 `scrapy.Item`
- 使用 `scrapy.Field()` 定義字段
- 字段名對應數據的鍵

## 在Spider 中使用

```python
from 項目名.items import 類名

def parse(self, response):
    item = 類名()
    item["字段名"] = 提取的數據
    yield item
# 或
def parse2(self, response):
    item = 類名(
        字段名=提取的數據
    )
    yield item
```

## 數據流向

```
Spider (創建並賦值item) → yield item → Pipeline (接收處理)
```

## 重點說明

- `scrapy.Field()`: 聲明一個數據字段
- Item 實例化後像字典一樣賦值
- 必須 `yield` 或 `return` 才能傳遞給pipeline
- 字段名需與 `items.py` 中定義一致
