# Spider 爬蟲代碼寫法

## 基本結構

```python
import scrapy

class XiaoSpider(scrapy.Spider):
    name = "xiao"  # 爬蟲名字
    allowed_domains = ["4399.com"]  # 被允許的域名
    start_urls = ["https://www.4399.com/special/1.htm"]  # 起始域名

    def parse(self, response):
        pass
```

## 數據提取方式

### 一次提取數據

```python
# 給了選擇器, 還未提取內容
txt = response.xpath("//div[@class='d_cen']/ul/li/p/a/text()")

# 提取內容, 返回列表
txt = response.xpath("//div[@class='d_cen']/ul/li/p/a/text()").extract()
```

### 分步提取（推薦）

```python
li = response.xpath("//div[@class='d_cen']/ul/li")
for contents in li:
    name = contents.xpath("./p/a/text()").extract_first()  # 提取第一項內容, 無內容返回None
    time = contents.xpath("./p/text()").extract_first()

    dic = {
        "name": name,
        "time": time,
    }
    yield dic  # 通過yield 返回數據給管道, 調用一次yield 返回一次數據並重新進入循環
```

## 重點說明

- `extract()`: 提取所有匹配的內容, 返回列表
- `extract_first()`: 提取第一項內容, 無內容返回 `None`
- `yield`: 返回數據給管道（pipeline）, 調用一次 yield 返回一次數據並重新進入循環
