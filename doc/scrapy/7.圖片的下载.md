# 圖片的下载

## 使用 ImagesPipeline

Scrapy 提供 `ImagesPipeline` 自動處理圖片下載,無需手動處理

## 配置圖片存儲

在 `settings.py` 中:

```python
ITEM_PIPELINES = {
    "項目名.pipelines.圖片管道類": 400,
}
IMAGES_STORE = "./images"  # 圖片保存路徑
```

## 自定義管道

繼承 `ImagesPipeline` 並重寫三個方法:

```python
from scrapy.pipelines.images import ImagesPipeline
import scrapy

class 類名(ImagesPipeline):
    def get_media_requests(self, item, info):
        # 負責下載圖片
        return scrapy.Request(item["圖片字段"])

    def file_path(self, request, response=None, info=None, item=None):
        # 設置文件保存路徑和文件名
        return f"./img/{文件名}.jpg"

    def item_completed(self, results, item, info):
        # 下載完成後處理,results 包含結果
        print(results)
```

## 重點說明

- 繼承 `ImagesPipeline` 後需重寫3個方法
- `get_media_requests`: 返回 `scrapy.Request` 發起下載
- `file_path`: 返回圖片保存的相對路徑
- `item_completed`: 下載完成後執行,可獲取下載結果
- `IMAGES_STORE` 在 `settings.py` 中配置圖片根目錄
