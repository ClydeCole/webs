# 創建項目
使用 `scrapy startproject games` 創建一個名爲`games` 的scrapy 項目

創建後的得到以下目錄結構  
```
games/
├── games/
│   ├── __init__.py
│   ├── items.py
│   ├── middlewares.py
│   ├── pipelines.py
│   ├── settings.py
│   └── spiders/
│       └── __init__.py
└── scrapy.cfg
```

各文件作用
- `__init__.py` 空文件, 標記 `games/` 為 `Python` 包
- `items.py` 定義爬取數據的結構
- `middlewares.py` 中間件
- `pipelines.py` 數據管道模板, 用於對爬取到的 `item` 進行後續處理
- `settings.py` 項目配置文件
- `spiders\` 存放爬蟲文件

# 創建爬蟲文件
`scrapy genspider xiao 4399.com`  
創建一個名爲 `xiao` 的爬蟲  
限定範圍爲 `4399.com`
