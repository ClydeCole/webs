# Pipeline 的使用

## 基本結構

```python
from itemadapter import ItemAdapter

class GamesPipeline:
    def process_item(self, item, spider):
        # process_item: 固定函數
        # item: 從spider通過yield傳來的數據
        # spider: 當前運行的爬蟲實例
        return item  # 必須返回item, 否則後續pipeline無法接收到數據
```

## 啓用Pipeline
Pipeline 默認不生效  
在 `settings.py` 中配置 `ITEM_PIPELINES`（數值越小優先級越高, 數值範圍0-1000）：

```python
ITEM_PIPELINES = {
    "games.pipelines.GamesPipeline": 300,
    "games.pipelines.NewPipeline": 299,
}
```

## 數據流向

```
Spider (yield item) → Pipeline (process_item) → 輸出/存儲
```

## 多個管道

項目中有 `GamesPipeline` 和 `NewPipeline` 兩個管道  
`ITEM_PIPELINES` 配置多個時按優先級順序執行  
數據流向：`Spider → GamesPipeline → NewPipeline → 輸出`  
每個管道必須返回 item
