# 前言介绍
### 字数与行数  
因为作文规定当中 字数必须正好在第1600字的那一行结束,  
不可多写也不可少写.  
因此必须在写作是严格按照[模板](doc/策划作文/note/模板.md)及本篇讲解的格式写作

资金来源部分可填写1-3 行内容 *具体视字数而定*,  
如果字数正常应该写入2 行

如果出现字数多写, 请在其他地方少写一行  
如果出现字数少写, 可在最后资金来源部分多填写内容

考试时用纸格式为 (往年皆是如此)
- 长 25 格
- 宽 24 格

### 表格与标题

**表格讲解**
- 标题 => 标题行数
- 实际 => 文本实际长度 *不含空着的部分*
- 内容 => 文本所占行数 *含空着的部分 一整行*
- 共计 => 含标题所占行数 *一整行*
- 字数 => **共计**所达到的字数 *一整行*

**标题讲解**

顶格标题:  
一二三...

标题:  
1.2.3...  
(1)(2)(3)...

*顶格标题与标题在表格内都称为标题*
