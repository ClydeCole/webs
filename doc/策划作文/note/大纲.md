# 一、活动目的

# 二、市场分析
## 1. SWOT 分析
(1)优势S: 你哪里好  
(2)劣势W: 你哪里不行  
(3)机会O: 别人的哪些对你有利  
(4)威胁T: 别人的哪一些对你不利

## 2. 目标群体分析
年龄, 性别, 购买特点(购买时关注什么), 受什么影响, 经常在哪里买

## 3. 产品定位分析
产品特点, 使用场合, 价格路线
