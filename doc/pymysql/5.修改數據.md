# 修改數據

## update 語句

修改數據使用 SQL 的 `update` 語句

需要指定表名、要修改的字段和新值, 以及篩選條件

```sql
update stu set name = '胡辣湯', score = 15 where sno = 11
```

## where 條件的重要性

`update` 如果不加 `where` 條件, 會修改表中的**所有**數據

這是一個非常常見且代價極高的錯誤

務必在執行 update 之前再三確認 where 條件是否正確

一個好的習慣是先用 `select` 查詢確認條件匹配的數據是否正確, 再執行 update

## 獲取影響行數

`execute` 方法的返回值是受影響的行數

通過這個返回值可以判斷 update 是否真的修改了數據

```python
count = cursor.execute(sql)
print(count)  # 輸出了幾條數據被修改
```

如果返回值是 0, 說明 where 條件沒有匹配到任何數據

## 事務要求

update 是修改操作, 同樣需要提交事務

不提交的話, 修改不會真正生效

```python
cursor.execute(sql)
conn.commit()
```

如果出錯也要回滾

## 參數化更新

和 insert 一樣, update 也應該使用參數化查詢

```python
sql = "update stu set name = %s, score = %s where sno = %s"
cursor.execute(sql, ("胡辣湯", 15, 11))
```

參數順序要和 SQL 中的 `%s` 一一對應

這樣做既能防止 SQL 注入, 也讓代碼更清晰
