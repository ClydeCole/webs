# 新增數據

## insert 語句

新增數據使用SQL 的 `insert into` 語句

需要指定表名、字段名和對應的值

```sql
insert into stu(name, gender, age, class, score) values ('值1', 1, 16, 2, 64)
```

字段順序與值順序必須一一對應

## 在Python 中執行

將SQL 語句作爲字符串傳遞給 `execute` 方法即可

如果 SQL 較長, 可以使用括號隱式換行, 提高可讀性

```python
sql = ("insert into stu(name, gender, age, class, score) values "
       "('小C', 1, 16, 2, 64)")
cursor.execute(sql)
```

## 獲取新增數據的id

插入數據後, 如果需要獲取這條數據的自增 id, 可以通過 `lastrowid` 獲取

這個屬性在執行完 `execute` 後就能訪問, 無需額外查詢

```python
cursor.execute(sql)
new_id = cursor.lastrowid
```

注意 `lastrowid` 只有在插入操作後才有意義, 查詢或更新操作時它的值不準確

## 新增的完整步驟

新增數據是寫操作, 必須提交事務

標準流程是先執行 SQL, 然後提交事務

如果出錯, 則回滾

```python
try:
    cursor.execute(sql)
    conn.commit()
except:
    conn.rollback()
```

## 使用參數化查詢

實際開發中不建議直接用字符串拼接 SQL, 應該使用參數化查詢

參數化查詢可以防止 SQL 注入攻擊, 同時也能自動處理引號轉義等問題

```python
sql = "insert into stu(name, age) values (%s, %s)"
cursor.execute(sql, ("張三", 18))
```

pymysql 使用 `%s` 作爲佔位符, 第二個參數傳入對應值的元組或列表

這樣做更安全, 也比拼字符串更簡潔
